Разница HTTP и HTTPS

HTTP (HyperText Transfer Protocol) – протокол передачи гипертекста. Используется при пересылке веб-страниц с одного компьютера на другой. HTTPS (HyperText Transfer Protocol Secure) – тоже протокол для передачи гипертекста, но использует дополнительное шифрование данных для более безопасной передачи информации.

Теги и Атрибуты

Что такое CSS?

CSS (Cascading Style Sheets) — это каскадные листы стилей, которые применяются для описания внешнего вида веб-документа

А HTML?

С помощью HTML появляется структура документа, а CSS — это уже его оформление
